# Ping Pong

A ping pong game implemented in Rust using Raylib.

## Building the project

### Requirements

- [CMake](https://cmake.org/)
- [Raylib library](https://www.raylib.com/)

### Build step

```sh
cargo build --release
```

### Installing the game

```sh
cargo install --path . --locked
```
